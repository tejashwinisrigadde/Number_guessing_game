## Number_guessing_game
   A simple python game where computer generates a random number and user tries to get it.
## Features
- Random number generation
- unlimited attempts
- input validation
- Attempt counter
##Technologies used
  python 3
##Futute Upgrades
  - difficult level(easy , medium ,high)
  - limited number of attempts
  - High score system
  - play again or exit option
